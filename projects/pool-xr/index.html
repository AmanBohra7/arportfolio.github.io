Introducing PoolXR

PoolXR is a mixed reality-based project that uses custom-trained object detection (for cue and game balls) along with the new Meta Passthrough Camera API. It brings the classic game of pool to life by infusing it with virtual information and guidance, combining real-world interaction with virtual elements, natural hand gestures, and intuitive gameplay.

The Passthrough Camera API allows seamless integration between the real and virtual worlds, making the game feel natural and engaging.

Key Features

My innovative approach to mixed reality pool gaming includes several groundbreaking features:

  • Real-time camera passthrough for natural interaction with the physical environment
  • Advanced physics simulation for realistic ball movement and collisions
  • Intuitive controls that mimic real pool cue movements
  • Multiplayer support for competitive gameplay

Technical Innovation

The core of PoolXR's success lies in its sophisticated use of camera passthrough technology. By leveraging advanced computer vision algorithms and real-time processing, I've created a system that:

  • Seamlessly blends real and virtual elements for an immersive experience
  • Maintains accurate spatial awareness for precise gameplay
  • Adapts to different lighting conditions and environments
  • Provides low-latency feedback for responsive gameplay
